Equations and Text Blocks

The Transformer model, introduced in "Attention Is All You Need", changed the landscape of NLP by replacing recurrent units with self-attention mechanisms.

Self-Attention Mechanism

At the core of the Transformer is the Scaled Dot-Product Attention. For a given set of queries $Q$, keys $K$, and values $V$, the attention is calculated as:

Where $d_k$ is the dimensionality of the keys. This scaling factor helps prevent the dot products from growing too large in magnitude, which would push the softmax into regions with extremely small gradients.

$$\text{MultiHead}(Q, K, V) = \text{Concat}(\text{head}_1, \dots, \text{head}_h)W^O$$ $$\text{This is a plain math block without background styling.}$$
Note: Multi-Head Attention allows the model to jointly attend to information from different representation subspaces at different positions.

Implementation Snippet

Here is a simplified implementation of the attention head in Python:

import torch
import torch.nn.functional as F

def scaled_dot_product_attention(q, k, v, mask=None):
    d_k = q.size(-1)
    scores = torch.matmul(q, k.transpose(-2, -1)) /  math.sqrt(d_k)
    
    if mask is not None:
        scores = scores.masked_fill(mask == 0, -1e9)
    
    attention_weights = F.softmax(scores, dim=-1)
    return torch.matmul(attention_weights, v), attention_weights

Conditional Entropy Calculation

Let's calculate the conditional entropy for a given feature $X$ over classes $Y$.

$$H(Y|X) = - \sum_{i} P(x_i) \sum_{j} P(y_j|x_i) \log_2 P(y_j|x_i)$$
