Drop your photos in this folder to have them show up in the scrolling filmroll
at the bottom of the site — no code editing required.

HOW TO NAME THEM
Name your files as plain numbers, starting at 1, using any of these
extensions: jpg, jpeg, png, webp (lowercase extensions only).

  1.jpg
  2.jpg
  3.png
  4.webp
  ...

RULES
- Start at 1 and don't skip numbers — the site stops looking as soon as it
  can't find the next number in the sequence. So 1.jpg, 2.jpg, 4.jpg (missing
  3) will only show photos 1 and 2.
- Any photo size/aspect ratio works — they're cropped to fit the frame.
- After adding or renaming photos, just refresh the page.
